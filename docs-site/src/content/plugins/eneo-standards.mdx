# eneo-standards

The standards plugin provides the runtime enforcement layer for the harness.

## What it owns

- TDD phase gate
- Bash firewall
- protected-file checks
- wave barrier
- ratchet checks
- pyright stop hook
- helper binaries such as `eneo-doctor-report`, `eneo-task-init`, and `eneo-task-update`
- the optional status line

Most developers do not call this plugin directly. You install it so the rest of the harness behaves safely and consistently.
