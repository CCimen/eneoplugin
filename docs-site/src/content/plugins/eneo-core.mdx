# eneo-core

The core workflow plugin. It contains the `/eneo-*` command set, the implementation subagents, and the Eneo-specific skills that shape planning and execution.

## Main commands

- `/eneo-core:eneo-new`
- `/eneo-core:eneo-discuss`
- `/eneo-core:eneo-plan`
- `/eneo-core:eneo-start`
- `/eneo-core:eneo-verify`
- `/eneo-core:eneo-ship`
- `/eneo-core:eneo-recap`
- `/eneo-core:eneo-doctor`
- `/eneo-core:eneo-prune`

## What it owns

- Fast / Standard / Deep lane triage
- phase execution and resume flow
- milestone closeout
- TDD writer/impl agents
- FastAPI, SvelteKit, tenancy, audit, and migration specialists

Use this plugin when you want the full harness workflow, not a single standalone helper.
