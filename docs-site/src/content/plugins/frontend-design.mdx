# Frontend Design

Create distinctive, production-grade frontend interfaces with high design quality.

## Why

AI-generated UIs tend to look generic — the "AI slop" aesthetic. This plugin provides design guidelines that push Claude toward bold, intentional design choices.

## Setup

```bash
/plugin install frontend-design@eneoplugin
```

## Usage

```
/frontend-design
```

Invoke when building web components, pages, dashboards, or any web UI.

## What it emphasizes

- **Typography** — Intentional font choices and hierarchy
- **Color themes** — Cohesive palettes, not random colors
- **Motion** — Purposeful animations and transitions
- **Spatial composition** — Thoughtful layout and whitespace
- **Visual details** — Borders, shadows, gradients used with intent

## When to use

- Building new UI components
- Redesigning existing pages
- Creating dashboards or data visualizations
- Any task where visual quality matters
