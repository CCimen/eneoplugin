## How plugins work

Each plugin can include:

- **Skills** — Slash commands that instruct Claude on what to do (e.g., `/finding list`)
- **Hooks** — Scripts that run automatically on events (e.g., checker runs after editing Python files)
- **MCP Servers** — Tool servers that provide custom capabilities (e.g., `typecheck()`)
- **CLAUDE.md** — Guidelines automatically loaded into every conversation
