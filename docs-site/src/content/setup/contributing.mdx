# Contributing

Want to add a new plugin or improve an existing one? Here's how.

## Plugin structure

Each plugin lives in `plugins/<name>/` and follows this structure:

```
plugins/my-plugin/
├── .claude-plugin/
│   └── plugin.json          # Plugin metadata (name, description, version)
├── skills/
│   └── my-plugin/
│       └── SKILL.md          # Skill definition (slash command)
├── hooks/                    # Optional: hook scripts
│   └── hooks.json
├── mcp/                      # Optional: MCP server
│   └── server.py
└── CLAUDE.md                 # Optional: auto-loaded guidelines
```

### Key rules

- The **skill folder name** must match the **plugin name** in `plugin.json` for the slash command to resolve correctly
- The `name` field in `SKILL.md` frontmatter must also match
- Bump the `version` in `plugin.json` when making changes to bust the cache

### plugin.json

```json
{
  "name": "my-plugin",
  "description": "Short description of what the plugin does",
  "version": "1.0.0"
}
```

### SKILL.md

```markdown
---
name: my-plugin
description: When and how to use this plugin. This text appears in the skills list.
---

# My Plugin

Instructions for Claude on how to execute the skill...
```

## Registering the plugin

Add an entry to `.claude-plugin/marketplace.json`:

```json
{
  "name": "my-plugin",
  "source": "./plugins/my-plugin",
  "description": "Short description"
}
```

## Development workflow

1. Fork the repo or create a branch
2. Create your plugin directory and files
3. Add the entry to `marketplace.json`
4. Test locally:
   ```bash
   /reload-plugins
   /my-plugin  # Test your skill
   ```
5. Submit a PR

## Tips

- **SKILL.md is a prompt**, not code. Write clear instructions for Claude to follow.
- Use `disable-model-invocation: true` in SKILL.md frontmatter if the skill only runs bash commands (no AI reasoning needed).
- Keep skills focused — one plugin should do one thing well.
- Reference the [checker](/plugins/checker) and [finding](/plugins/finding) plugins as examples of different plugin types.
