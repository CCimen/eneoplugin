# eneoplugin

A Claude Code plugin marketplace with AI team tools, guidelines, and integrations for Eneo developers.

## What is this?

**eneoplugin** is a shared collection of [Claude Code](https://claude.com/claude-code) plugins built by and for the Eneo AI team. Each plugin adds specialized capabilities — from type checking and code review to project management and design guidelines.

Plugins are installed via Claude Code's marketplace system and activated with slash commands.

## Quick start

```bash
# 1. Add the marketplace
/plugin marketplace add CCimen/eneoplugin

# 2. Install the plugins you need
/plugin install finding@eneoplugin
/plugin install checker@eneoplugin
/plugin install karpathy-guidelines@eneoplugin
```

See [Getting Started](/setup) for detailed setup instructions.
